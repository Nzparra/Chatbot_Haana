# Chatbot_Haana
 Chat for Haana App project

- Feed Forward Neural net with 2 hidden layers.

## Installation

### Create an environment
```console
mkdir myproject
$ cd myproject
$ python3 -m venv venv
```

### Activate it
Linux:
```console
. venv/bin/activate
```
Windows:
```console
venv\Scripts\activate
 ```

### Install Dependencies

 ```console
pip install nltk
 ```
Run this once in your terminal:
 ```console
$ python
>>> import nltk
>>> nltk.download('punkt')
```

## Usage
Run
```console
python train.py
```
```console
python chat.py
